# Officebite - Kampala Office Food Delivery

A modern, responsive React + Vite application for ordering breakfast and lunch in Kampala buildings.

## Features
- **PWA Support**: Installable on mobile and works offline.
- **Hero Carousel**: Beautifully animated slides for Breakfast, Lunch, and Delivery.
- **Modern UI**: Smooth transitions with Framer Motion and premium aesthetics.
- **Menu Management**: Selection for breakfast items (teas, snacks) and lunch options with pricing.

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Assets Setup**:
   I have generated high-quality images for you. Please move them from the workspace to a new folder: `src/assets/`.
   - Rename `breakfast_hero_...png` to `breakfast_hero.png`
   - Rename `lunch_hero_...png` to `lunch_hero.png`
   - Rename `delivery_hero_v2_...png` to `delivery_hero.png`

3. **Run Dev Server**:
   ```bash
   npm run dev
   ```

## Tech Stack
- **React 18**
- **Vite**
- **Framer Motion** (Animations)
- **Lucide React** (Icons)
- **Vanilla CSS**
